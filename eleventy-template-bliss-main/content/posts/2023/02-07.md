Outre une introduction sur la structure et l'histoire du Web, cette première séance a traité de la sémantique et de la syntaxe du HTML, le langage de balisage qui structure les contenus du Web. Cliquez <a href="/images/slides-export.pdf" download="slides">ICI</a> pour télécharger les diapositives de la séance.

Après un aperçu rapide des principales [balises proposées par HTML](https://web.stanford.edu/group/csp/cs21/htmlcheatsheet.pdf), nous avons examiné de plus près l'insertion de certains éléments dans une page web. En particulier, nous avons observé :

### La structure de base d'un document HTML

```html
  <!DOCTYPE html>
  <html lang="fr">
  <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Mon site</title>
      </head>
  <body>
      
  </body>
  </html>
```


### L'insertion d'un favicon dans les métadonnées

```html
  <link rel="icon" type="image" href="chemin-vers-l-image.ico">
```

### L'insertion d'images

```html
<link rel="icon" type="image" href="chemin-vers-l-image.ico">
``` 

### L'insertion de liens

```html
<a href="LIEN">
```

### L'insertion d'images avec liens

```html
<a href="LIEN"> 
  <img src="chemin-vers-l-image" alt="description" width="30%" height="30%">
</a>
```



### L'insertion de boutons avec liens

```html
<button type="button">Description<a href="LIEN" target="blank"></button>
```
OU

```html
<a href="LIEN"><button>Description</button></a>
```

### L'insertion de tableaux

```html
    <table border="1">
        <tr>
            <th>Instructrice</th>
            <th>Heure</th>
            <th>Local</th>
        </tr>
        <tr>
            <td>Giulia Ferretti</td>
            <td>9h30 - 15h30</td>
            <td>c-2035</td>
        </tr>
    </table>
```

### L'insertion de formulaires

  ```html
    <form action="results.html" method="GET">
        <div>
            <label>Nom et Prénom</label>
            <input name="name">
        </div>
        <div>
            <label>Password</label>
            <input name="password">
        </div>
        <button>Submit</button>
    </form>
  ```

N.B.: Pour afficher les données saisies après l'envoi du formulaire, vous devez enregistrer le fichier <a href="/images/results.html" download="results.html">results.html</a> dans le même dossier que le formulaire.

### Structurer un CV

```html
  <!DOCTYPE html>
  <html lang="fr">
  <head>
    <meta charset="UTF-8">
    <title>Curriculum vitæ</title>

  </head>
  <body>
    <!-- En-tête -->
    <header class="entete">
      <!-- Bonne pratique: un seul élément `h1` par document -->
      <h1 class="entete__nom">Giulia Ferretti</h1>

      <span class="entete__titre">Doctorante en humanités numériques</span>
    </header>

    <section>
      <h2 id="informations-personnelles">Informations personnelles</h2>

      <dl class="liste-infos-personnelles">
        <dt>Date de naissance</dt>
        <dd>01 janvier 0000</dd>

        <dt>Adresse</dt>
        <dd>Montréal, QC</dd>

        <dt>Téléphone</dt>
        <dd>+1 (234) 456-7890</dd>

        <dt>Courriel</dt>
        <dd>bonjour@exemple.com</dd>

        <dt>Site web</dt>
        <dd>http://mapageweb.org/</dd>
      </dl>
    </section>

    <!-- Une règle horizontale (élément auto-fermant) -->
    <hr>

    <!-- Colonne principale, corps du document -->
    <section>
      <h2 id="profil">Profil</h2>

      <p>Giulia Ferretti est doctorante
         à l’université de Montréal. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nullam ac tempus ante. Maecenas efficitur tempus consequat. Aenean egestas ligula sit amet ligula consequat facilisis. Etiam vel vulputate nulla. Integer eget fermentum orci, ac sagittis libero. Etiam efficitur volutpat neque, id placerat ante porta vel.</p>
    </section>

    <hr>

    <section>
      <!-- Bonnes pratiques: un identifiant unique pour les en-têtes -->
      <h2 id="education">Éducation</h2>

      <!-- Chaque entrée est balisée dans une section -->
      <!-- Première entrée -->
      <section class="entree">
        <!-- Le nom de l’entrée dans un en-tête de niveau 3 -->
        <h3 class="nom">Université X</h3>

        <!-- Pour isoler la période -->
        <div class="periode">depuis 2020</div>

        <!-- Type de diplôme -->
        <div class="diplome">Doctorat</div>

        <div class="description">Lorem ipsum dolor sit amet.</div>
      </section>

      <!-- Deuxième entrée -->
      <section class="entree">
        <h3 class="nom">Université Y</h3>

        <div class="periode">2018–2020</div>
        <div class="diplome">Maîtrise</div>
        <div class="description">Lorem ipsum dolor sit amet.</div>
      </section>

      <!-- Troisième entrée -->
      <section class="entree">
        <h3 class="nom">Université Z</h3>

        <div class="periode">2015–2018</div>
        <div class="diplome">Doctorat</div>
        <div class="description">Lorem ipsum dolor sit amet.</div>
      </section>
    </section>

    <hr>

    <section>
      <h2 id="experience-professionnelle">Expérience professionnelle</h2>

      <!-- Première entrée -->
      <section class="entree">
        <h3 class="nom">Employeur X</h3>

        <div class="periode">2015–2018</div>

        <div class="description">Lorem ipsum dolor sit amet.</div>
      </section>

      <!-- Deuxième entrée -->
      <section class="entree">
        <h3 class="nom">Employeur Y</h3>

        <div class="periode">2015–2018</div>

        <div class="description">Lorem ipsum dolor sit amet.</div>
      </section>

      <!-- Troisième entrée -->
      <section class="entree">
        <h3 class="nom">Employeur Z</h3>

        <div class="periode">2015–2018</div>

        <div class="description">Lorem ipsum dolor sit amet.</div>
      </section>
    </section>
    <hr>

    <section>
      <h2 id="prix-distinctions">Prix et distinctions</h2>

      <!-- Première entrée -->
      <section class="entree">
        <h3 class="nom">Prix X</h3>

        <div class="periode">2019</div>

        <div class="description">Lorem ipsum dolor sit amet.</div>
      </section>

      <!-- Deuxième entrée -->
      <section class="entree">
        <h3 class="nom">Bourse FRQSC</h3>

        <div class="periode">2022</div>

        <div class="description">Lorem ipsum dolor sit amet.</div>
      </section>
    </section>

    <hr>

    <!-- Un autre bloc secondaire (souvent à la fin du CV) -->
    <section>
      <h2 id="competences">Compétences</h2>

      <!-- Une liste non ordonnée -->
      <ul>
        <li>Une lise</li>
        <li>non ordonnée</li>
        <li>de competences</li>
      </ul>
    </section>

    <!-- Pied de page -->
    <footer>
      Pied de page – Giulia Ferretti
    </footer>
  </body>
  </html>
```
